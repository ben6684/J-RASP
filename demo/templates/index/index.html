J-RASP Purpose

J-RASP is a journal publishing reproductible algorithms for sound processing. Every publication contains a detailed description and implementation of the algorithm and a demo program. Every submission is peer-reviewed (including source code and demo program) and we certify that the implementation fits the description. The algorithms submitted to J-RASP can be original in the methodology or in the implementation. They may also consist in reviews of existing methods.

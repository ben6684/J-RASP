Article Guideline

  1. The article must describe all steps of the algorithm. Some steps can be described as pseudo code
  2. The article must contain:
    • an abstract
    • an introduction
    • a scientific description of the method
    • a discussion about the performances of the algorithm on several examples
    • an investigation on the influence of the parameters
    • some bibliographical references

Software guideline

  1. The submitted code must be a straighforward implementation of the algorithm described in the article
  2. The code should be commented and readable
  3. The languages accepted are : Matlab, C/C++, Python 2.x, Python 3.x. Please only use standard toolboxes and librairies. We warn you that the acceptation of toolboxes, librairies and versions is let to the appreciation of the editor
  4. The archive file (zip, tar) should include:
    • a README file describing compilation, installation, execution, versions, toolboxes, librairies and dependencies
    • the source code (if C/C++, please provide a make or cmake)
    • a demo folder containing demo data and demo source code (at least 2 examples)
